### 1. Abstract Factory

> Program an interface, not an implementation

| Factory Method        | Abstract Factory                          |
|-----------------------|-------------------------------------------|
| Create objects        | Create factories *(Factory of factories)* |
| Method -> single type | Object -> Families of objects             |
| Inheritance           | Composition                               |

#### Participants

- Abstract product
- Concrete product
- Abstract factory
- Concrete factory
- Client
